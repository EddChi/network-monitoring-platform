# Distributed Network Monitoring Platform

A full-stack network monitoring system that collects real-time telemetry from multiple agents and displays network health through a central dashboard.

## Purpose

This project is being built to demonstrate:

- Networking concepts
- Backend development
- Distributed systems
- Databases
- Containerisation
- Real-time communication
- System monitoring

## Planned Features

- Monitoring agents
- Spring Boot REST API
- PostgreSQL database
- Real-time dashboard
- WebSocket updates
- Alert system
- Network latency checks
- Docker deployment

## Tech Stack

- Java 21
- Spring Boot
- PostgreSQL
- Docker
- React
- WebSockets
- GitHub Actions
